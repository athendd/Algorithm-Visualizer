Selection Sort

A comparison-based algorithm. Sorts an array be repeatedly selecting the smallest or largest element from the unsorted section of the given list and swapping it with the first unsorted element.

Selection Sort Steps:

1. Find the smallest element and swap it with the first element.

2. Find the smallest element among the remaining elements and swap it with the second element

3. Repeat until the list is sorted. 

      
procedure selectionSort(array)
  for i <- 0; i < length(array); i++ do
    min_index <- i

    for j <- i + 1; j  < length(array); j++ do
      if array[j] < arr[min_index] then
        min_index <- j
      end if
    end for

    swap array[i] with array[min_index]
  end for
end procedure
