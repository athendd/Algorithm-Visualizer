Insertion Sort

A comparison-based sorting algorithm. It works similarly to how we sort playing cards in our handsâ€”by taking one card at a time and inserting it into its correct position relative to the already sorted portion of the hand.

Insertion Sort Steps:

1. Divide given list into sorted and unsorted part

2. The first element is labeled as sorted

3. Picks an element from unsorted part of list and inserts it into its correct position in the sorted part of the list by shifting elements

4. Process is repeated until all elements are sorted 

      
procedure insertionSort(array)
  for i <- 1 to length(array) - 1 do
    key <- array[i]
    j <- i - 1
    while j >= 0 and array[j] > key do
      array[j + 1] <- array[j]
      j <- j - 1
    end while
    array[j + 1] <- key
  end for
end procedure
