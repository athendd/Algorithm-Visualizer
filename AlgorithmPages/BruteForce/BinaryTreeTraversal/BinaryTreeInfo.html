Binary Tree Traversal Inorder

A binary tree is a hierarchical data structure in which each node has at most two children, typically referred to as: left child, right child. Tree traversal refers to the process of visiting every node in a tree exactly once in a specific order.


For in-order traversal, you visit:

  1. The left subtree

  2. The root node

  3. The right subtree
      
procedure inorderTraversal(root)
  result <- []

  procedure traverse(node)
    result.push(node)
    if node.left then
      traverse(node.left)
    if node.right then
      traverse(node.right)
  end procedure

  traverse(root)
  return result
end procedure
