Maximum Sum Path

The Maximum Sum Path algorithm is most often used in the context of binary trees and aims to find the path with the largest sum of node values, whereby a path can start and end at any node, but each node can only be used once per path. If the input is a binary tree, then the Maximum Sum Path algorithm will make use of the DFS algorithm within itself to traverse the tree and calculate the maximum sum of node values from a path. This makes the Maximum Sum Path algorithm something of a modification of the DFS algorithm, if not an extension to it.

The psuedocode and middle animation panels depict a Maximum Sum Path algorithm made for binary tree inputs, however, with a few modifications, the Maximum Sum Path algorithm can work with 2D arrays and matrices too.

Maximum Sum Path Steps:

1. Base Case: If the current node is null, return 0.

2. Recursively Call: Recursively calculate the maximum path sum for the left and right subtrees.

3. Calculate Path Sums:
3.1. Calculate Left: Calculate the maximum path sum that includes the current node and its left subtree (if it exists and has a positive contribution).
3.2. Calculate Right: Calculate the maximum path sum that includes the current node and its right subtree (if it exists and has a positive contribution).
3.3. Calculate Both: Calculate the maximum path sum that includes the current node and both left and right subtrees (if they exist and have positive contributions).

4. Update Global Maximum: Update the overall maximum path sum (initially set to the minimum possible value) with the maximum of its current value and the newly calculated sums.

5. Return Value: Return the maximum path sum that includes the current node and either its left or right subtree (whichever contributes more to the overall path sum).

6. Final Result: The final value after recursion completes represents the maximum path sum from any sequence of nodes in the given binary tree. 

    
function maxPathSum(root):
    global maxPathSum = -infinity
    function findMaxPathSum(node):
        if node is null:  // Base case
            return 0
        leftSum = findMaxPathSum(node.left)
        rightSum = findMaxPathSum(node.right)
        currentPathSum = node.value + max(leftSum, rightSum)
        maxPathSum = max(maxPathSum, node.value + leftSum + rightSum)
        return max(0, currentPathSum)  // Ensure we don't return a negative value
    findMaxPathSum(root)
    return maxPathSum
