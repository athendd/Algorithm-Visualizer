Counting Sort

Counting Sort is a non-comparison-based sorting algorithm. It is particularly efficient when the range of input values is small compared to the number of elements to be sorted. The basic idea behind Counting Sort is to count the frequency of each distinct element in the input array and use that information to place the elements in their correct sorted positions.



Counting Sort Steps:



1. Find maximum element in given list



2. Create counting list with length equal to maximum element plus 1



3. Store count of each unique element in given list at their respective indices in the counting list



4. Modify the counting list to store the cumulative sum of each element



5. Iterate through the given list again, using the cumulative sums in the counting list to determine the correct position of each element in the sorted output list

      
procedure countingSort(arr)
  n <- length(arr)
  m <- 0

  for i <- 0; i < n; i++ do 
    m = max(m, arr[i])
  
  countarray <- [0] * (m + 1)

  for i <- 0; i < n; i++ do
    countarray[arr[i]]++
  
  for i <- 1; i <= m; i++ do
    countarray[i] += countarray[i - 1] 

  outputarray <- [] * n

  for i <- n - 1; i >= 0; i-- do
    outputarray[countarray[arr[i]] - 1] <- arr[i]
    countarray[arr[i]]--
  
  return outputarray
  
end procedure
