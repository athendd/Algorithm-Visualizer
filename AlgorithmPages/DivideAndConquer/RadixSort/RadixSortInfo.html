Radix Sort

A linear sorting algorithm that sequentially sorts elements based on the value of each digit/letter place. The algorithm initially compares the lowest place values for each element in the list and sorts them based on this. It then repeats this process for each ascending place value until the last place value is sorted on, which will ultimately produce a stably sorted list.

The psuedocode and middle animation panels depict a positive integers-only Radix Sort algorithm, however, with a few modifications, the Radix Sort algorithm can also be used to efficiently sort strings, as well as negative integers and decimals.

Radix Sort Steps:

1. Match Element Sizes: If elements do not all have the same number of value places, then add leading zeroes, or a character that lexicographically precedes the alphabet, to elements as necessary until all elements have the same number of place values as the element with the highest number of place values.

2. Sort By Place Value: Take the least significant place value of each element and sort them using counting sort logic before changing the order of elements to match the ordering of the isolated place values.

3. Sort All Place Values: Repeat Step 2 for each ascending place value until the list has been sorted by every place value. 

    
Algorithm: RadixSort(a[], n):
  max = a[0]
  for (i=1 to n-1):
    if (a[i]>max):
      max=a[i]
  For (pos=1 to max/pos>0):
    countSort(a, n, pos)
    pos=pos*10
