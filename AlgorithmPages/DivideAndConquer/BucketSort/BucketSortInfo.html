Bucket Sort

A sorting algorithm that divides the input array into a number of buckets. Each bucket is then sorted individually, either using a different sorting algorithm or recursively applying the bucket sort algorithm. Finally, the sorted buckets are concatenated to form the sorted array.

There are mainly three steps in the algorithm:

1. Divide into Buckets: Distribute the elements of the input array into a fixed number of buckets. The range of values in each bucket is determined based on the input data.

2. Sort Each Bucket: Sort the elements within each bucket. This can be done using another sorting algorithm (e.g., insertion sort) or recursively applying bucket sort.

3. Concatenate Buckets: Combine the sorted buckets to form the final sorted array.

Note: Bucket sort is most effective when the input data is uniformly distributed over a range. 

    
    procedure bucketSort(array, n)
      create n empty buckets
      for each element in array do
        insert element into the appropriate bucket
      end for
      for each bucket do
        sort the bucket
      end for
      concatenate all buckets into the original array
    end procedure
