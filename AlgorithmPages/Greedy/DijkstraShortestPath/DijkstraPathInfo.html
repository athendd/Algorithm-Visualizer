Dijkstra's Shortest Path

An algorithm designed to find the shortest path from one vertex to all other vertices. It does this by repeatedly selecting the nearest unvisited vertex and calculating the distance to all of the unvisited neighboring vertices. Broadly speaking, the algorithm is used for solving single-source shortest path problems for directed or undirected paths, exluding those with negative edges, and the output of the algorithm is often referred to as the shortest path spanning tree. By terminating the algorithm early, it is also possible to determine the shortest path to a specific vertex from the source vertex, as opposed to the entire shortest path spanning tree.

Dijkstra's Shortest Path Steps:

1. Choose A Source Vertex: Select one of the vertices in the graph to start from and set it as the current vertex, this will be the source vertex.

2 Choose New Vertex: Choose the unvisited vertex with the shortest distance from the current vertex to be the new current vertex (this will be itself if the current vertex is the source vertex).

3. Calulate Neighbor Distances: For each of the current vertex's unvisited neighbor vertices, calculate the distance from the source vertex and update the distance if the new calculated distance is lower.

4. Mark As Visited: After calculating all of the unvisited neighbor distances, mark the current vertex as visited to indicate that it doesn't need to be checked again.

5. Visit All Vertices: Repeat steps 2-4 until all vertices are visited. The end result will be the shortest path from the source vertex to every other vertex in the graph, i.e. the shortest path spanning tree. 

    
      function Dijkstra(Graph, source):
        for each vertex v in Graph.Vertices:
          dist[v] ← INFINITY
          prev[v] ← UNDEFINED
          add v to Q
        dist[source] ← 0
        while Q is not empty:
          u ← vertex in Q with minimum dist[u]
          remove u from Q
          for each neighbor v of u still in Q:
            alt ← dist[u] + Graph.Edges(u, v)
            if alt < dist[v]:
              dist[v] ← alt
              prev[v] ← u
        return dist[], prev[]
